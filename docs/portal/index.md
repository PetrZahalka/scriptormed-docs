# ScriptorMed Portál

**ScriptorMed Portál** je webová aplikace dostupná na [app.scriptormed.com](https://app.scriptormed.com), kde spravujete svůj účet, předplatné a licenční klíče.

---

## Co portál nabízí

- **Registrace ordinace** — vytvoření účtu pro vaši praxi
- **Správa předplatného** — výběr plánu, změna, zrušení
- **Přehled využití** — kolik konzultací jste čerpali
- **Licenční klíče** — zobrazení a správa klíčů pro desktop aplikaci
- **Fakturace** — přehled plateb a faktur
- **Top-up kredity** — dokoupení konzultací nad rámec plánu

---

## Navigace

| Sekce | Co tam najdete |
|-------|---------------|
| **Přehled** | Stav předplatného, zbývající konzultace, aktivní zařízení |
| **Předplatné** | Aktuální plán, změna plánu, fakturační období |
| **Faktury** | Historie plateb, stažení faktur (PDF) |
| **Licenční klíče** | Vaše klíče, připojená zařízení |
| **Nastavení** | Údaje ordinace, kontaktní e-mail |
